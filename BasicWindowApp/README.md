# Raylib Rotating Cube Example

A simple C application that displays a rotating 3D cube using the raylib graphics library.

## Prerequisites

- [raylib](https://www.raylib.com/) library installed on your system
- C compiler (gcc/clang)
- Make

## Building

```bash
make
```

## Running

```bash
./cube_app
```

## Controls

- Use mouse to rotate the camera
- Escape key to exit
